### Run
```python
pytest test_scenario_0001.py
```

### Selenium WEB Driver Requirements
chromedriver in `$PATH` (for Chrome) 

geckodriver in `$PATH` (for Firefox)

### PyTest options
```python
--language  #  ru, en, fr, etc.
--browser_name  #  chrome, firefox
-m  # select test case. ex: tc_0001
```
